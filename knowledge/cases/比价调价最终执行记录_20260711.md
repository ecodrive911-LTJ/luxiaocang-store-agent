# 商品比价调价方案 — 最终执行记录

## 执行日期
2026-07-11

## 任务概述
对鹿小仓两店（广安店、财富店）与竞品店（小柴购、丽晨超市）进行商品比价，按"竞品价-0.05元"规则生成调价方案。经多轮规格校验和复核重审，生成最终可执行调价清单。

## 执行过程

### v1: 模糊匹配（初版）
- 方法：Jaccard重叠度(0.12) + SequenceMatcher(0.45)
- 结果：广安店557条匹配，财富店309条匹配
- 问题：仅按名称文本匹配，未校验规格/品牌/品类

### v2: 规格校验
- 新增：从品名和规格列提取克重/容量/数量，套装检测
- 结果：广安店166条通过（排除391），财富店131条通过（排除178）
- 问题：95+51=146条标注"需复核"

### v3: 需复核深度重审
- 品牌校验：双方品牌不同 → 排除
- 子品类校验：白酒vs啤酒、猫粮vs犬粮 → 排除
- 套装vs单品：比单价，差异过大 → 排除
- 竞品价格异常（<0.5元或量级差异）→ 排除
- 初步：广安店 CONFIRM=47 REJECT=45 MANUAL=3；财富店 CONFIRM=25 REJECT=21 MANUAL=5

### v3b: 二次过滤
- 竞品品名为通用词 → 排除
- 品名相似度<0.15 → 排除
- 价格差异极大且非同品牌 → 降级为MANUAL
- 避孕套同品类不同品牌 → 恢复为CONFIRM（消费者可替代）
- 最终：广安店 CONFIRM=20 REJECT=67 MANUAL=8；财富店 CONFIRM=10 REJECT=34 MANUAL=7

### v4: 生成最终调价方案
- 合并v2直接通过 + v3b确认可比 = 最终纳入调价
- 排除：误匹配 + 需人工判断

## 调价规则
- 鹿小仓售价 = 竞品售价 - 0.05元
- 若调价后低于采购价 → 标注"亏本预警"
- 排除规则：品牌不同/规格不一致/套装vs单品/品类不同/价格异常
