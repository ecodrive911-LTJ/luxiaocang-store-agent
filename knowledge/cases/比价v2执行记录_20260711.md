## 版本变更
v1 → v2 核心升级：
1. **规格校验**：克重/容积差>10g/ml → 排除
2. **套装检测**：品名或规格含 ×N/N瓶装/整箱/套装 → 排除；品名套装不一致 → 排除
3. **单位校验**：g vs ml vs count 跨单位 → 排除
4. **亏本预警**：调价后价格 < 采购价 → 标注"!!亏本"
5. **价格偏差熔断**：比值<0.5或>2.0 → 标注"需复核"
6. **方向标识**：↓降价 / ↑涨价 / —不变
